Likes & Dislikes


Likes

  1. Community
  2. Open Source
  3. Updates

Dislikes

  1. Travel
  2. Merchants

The community is helpful and nice even if the questions asked are obvious or simple.

The game is Open Source which is extremly nice especially for what type of game it is. Might be the only game that is open source in the same genre/style of game.

The game updates a lot, since I started playing on 2021-4-07 the game already has updated multiple times and it is only 2021-04-11 when writing.


Traveling is way to slow especially if you have a limited amount of time to play.

Merchants are way to hard to find in towns.
