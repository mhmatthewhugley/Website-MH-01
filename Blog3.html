Reasons for not using JS.

The reason(s) for why I try to avoid using JS is:

  1. Tracking.
  2. Bloat.
  3. Abused.
  4. Fail.
  5. Mainteance.
  6. Dependencys.

(JS stands for JavaScript)


Most websites including the one you are on right now uses JS for tracking stuff like what pages you go to. (We use Google Anaytics.)

Some websites make it so you cannot see any content unless you have js enabled, which is not right cause you should be able to choose if you want to be tracked or not.

As such I have made sure that all content on this website is viewable even if you have JS disabled.

JS is bloated and messy interms of performance and code.

JS is abused more then it should be.

JS can fail because of a online dependency changing its code or it could fail because of a simple coding mistake. Sometimes when it fails it can be devastating on a website.

JS requires more maintance than html when new updates come or when you want to add a simple feature compared to html.

JS can be dependent on a single url to a file and if that file is not hosted by the same company hosting the website then sometimes if the JS file goes down so does the website.


Reasons to use JS.

  • Looks.
  • Usability.

    • JS is used for improveing the look of a website and doing things that normally would not be possible with just html.

    JS is also used for function which is why some websites dont allow certain pages to function unless you have JS enabled.
